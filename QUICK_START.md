# Vysper Assistant - Quick Start Launcher Guide

You can now start the **Vysper Assistant** application directly by clicking on an icon, without using the terminal.

## How to Launch the Application

1. Open your File Explorer and navigate to the project directory:
   `D:\Phani\Tools\Playground\Vysper\Assistant`
2. Locate the file named **`Vysper.bat`** (it might just show as `Vysper` with a gears icon).
3. **Double-click** `Vysper.bat` to launch the application.
   - The command window will open and close instantly in a split second.
   - The application windows (the chat assistant and the overlay screen capture panel) will load and open on your screen.

---

## (Optional) Create a Desktop Shortcut Icon

If you want to open it from your Windows Desktop:

1. Right-click on **`Vysper.bat`** in the project directory.
2. Select **Show more options** -> **Send to** -> **Desktop (create shortcut)**.
3. Go to your desktop. You will see a new shortcut named `Vysper - Shortcut`.
4. (Optional) Right-click the shortcut, select **Rename**, and change it to **`Vysper`**.
5. (Optional) Customize the icon:
   - Right-click the shortcut -> select **Properties**.
   - Under the **Shortcut** tab, click **Change Icon...**.
   - Click **Browse...**, navigate to `D:\Phani\Tools\Playground\Vysper\Assistant\assests\icons\terminal.png` or any `.ico` file, click **OK**, and click **Apply**.
6. You can now start Vysper at any time simply by double-clicking this Desktop icon!
